# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an academic repository for **Orion Observatory** — a project observatory developed as part of a PhD-level course (TAES - Tópicos Avançados em Engenharia de Software), supervised by Prof. Ivaldir Honório de Farias Júnior.

The observatory focuses on **project observability**: monitoring, transparency, and providing insights about projects. It may also include supplementary information beyond core project metrics.

## Repository Structure

- `contexto/` — Reference materials
  - `apresentacoes/` — Course presentations (PDFs)
  - `artigos/` — Academic papers and theses on project observatories
  - `projetos/` — Real-world project documents (client projects in Portuguese)
- `atividades/` — Course assignments and deliverables

## Language

All content is in **Brazilian Portuguese**. Responses and generated content should default to Portuguese unless otherwise requested.

## Notes

- This is a document-based repository — no build system, tests, or application code.
- PDFs can be read with the Read tool (use the `pages` parameter for large files).
- `.docx` files in `contexto/projetos/` are client project case studies used as observatory data sources.
